# @fugood/node-llama-win32-arm64-qualcomm

Native module for llama.node targeting win32-arm64-qualcomm.

This package contains the pre-compiled native module for the specified platform and architecture with Qualcomm OpenCL and Hexagon support.

## Installation

This package is typically installed automatically as a dependency of `@fugood/llama.node`.

## Platform Support

- **OS**: win32
- **Architecture**: arm64
- **Variant**: qualcomm
- **Backends**: OpenCL, Hexagon NPU

## Usage

This package is not meant to be used directly. It is consumed by the main `@fugood/llama.node` package.
